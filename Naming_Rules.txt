1-TypeDefs:

u8 --> unsigned char
s8 --> signed char

u16 --> unsigned short int
s16 --> signed short int

u32 --> unsigned long int
s32 --> signed long int

f32 --> float
f64 --> double

------------------------------------------------------------------------------------------------------

2-VariableNames:

Drivers:
	Component_type+Description
	ex: ADC_u8Index --> Variable is an unsigned char in ADC layer and used as an index for array.

Input argument for functions:
			     Copy_type+Description
			     ex: void Func(s16 Copy_s16Result)

Application / In Functions:
			   Local/Global_type+Description
			   ex: Local_u8Counter / Global_f32Voltage

------------------------------------------------------------------------------------------------------

3-FunctionNames:

Component_returnType+Description
ex: void LCD_voidDisplayChar(u8 Copy_u8Char);
    u16 GPIO_u16Init(void);

------------------------------------------------------------------------------------------------------

4-Macros:

Component_Description (Should be capitalized --> uppercase)
ex: EEPROM_DATA_PORT
    LED_ACTIVE_HIGH

------------------------------------------------------------------------------------------------------

NOTE:
     Codes should be provided with descriptive comments as well as gitHub commits.
